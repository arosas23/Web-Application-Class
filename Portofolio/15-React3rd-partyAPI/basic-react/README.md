# Star Wars Movie Gallery

A React application showcasing Star Wars episodes with an interactive movie gallery. Browse iconic characters, view episode details, like/dislike movies, and leave comments.

## Project summary

- Frontend: Create React App in `src/` (React + react-router-dom). Entry: `src/index.js` -> `App.jsx`.
- Backend: Express server in `server/` that provides API endpoints and seeds MongoDB from `server/data.js`.
- Images: served from the frontend `public/images/` (CRA static public folder).
